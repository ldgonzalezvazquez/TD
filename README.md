## Repository organization

This repository contains the code developed as part of my PhD thesis and is organized into separate directories corresponding to the main methodological components of the work.

- **SRARec/**  
  Contains the code related to the detection and analysis of intra-host recombination from raw sequencing data.

- **SRASel/**  
  Contains the code used to study intra-host evolutionary dynamics and selection from longitudinal sequencing data.

- **RecSim/**  
  Contains the scripts and supplementary code developed for simulation-based analyses and benchmarking.

This organization is intended to keep the different lines of work clearly separated while preserving a common and reproducible computational structure across the thesis. Each directory contains the corresponding scripts, documentation, and supporting material required to understand the methodological implementation of that part of the work.

Only code and general documentation are included in this repository. Study-specific results, patient-level data, and other confidential or unpublished materials are intentionally excluded.

## Notes on simulations and licensing

The **RecSim/** directory includes the code developed for the simulation framework used in this thesis. However, it does **not** include the source code of **SANTA-SIM**, which was used as the basis for part of the simulation work, in order to avoid any potential licensing conflicts or redistribution issues.

A modified version of the simulation code can be made available upon request. Requests can be sent to: **luisdaniel.gonzalez@uvigo.es**.

## Availability and scope

This repository is intended to document the computational methodology underlying the thesis in a transparent and reproducible way. It is not intended to function as a complete distribution of all data, intermediate outputs, or collaboration-specific materials associated with the individual studies.
